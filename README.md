# Translate | Digital Native (Sidephone Edition)

A fast, API-driven translation Progressive Web App (PWA) designed specifically for the strict vertical constraints of the Sidephone and other minimalist displays.

## Why This Exists
Standard translation applications often utilize side-by-side layouts or floating conversational bubbles that break completely on a 2.8-inch, 3:4 aspect ratio screen. 

This translator is engineered as a strict vertical stack. It prioritizes a linear user flow—Input, Action, Output—ensuring that every element is fully visible and accessible without requiring horizontal scrolling or complex gestures. It provides essential travel and communication utility without the bloat of a full smartphone app.

## Features
- **Vertical Stack Architecture:** A focused layout built perfectly for 480x640 resolutions.
- **Curated Selection:** Pre-loaded with 12 of the most common languages to prevent overwhelming tiny screens with endless dropdown menus.
- **API-Driven:** Utilizes the free MyMemory Translation API via native fetch requests. No backend servers or API keys required.
- **Quick Copy:** Built-in clipboard integration to quickly copy translated text for SMS or email.

## Live Demo & Installation
You can use the translator directly in your browser:
**[Launch Translate | Digital Native](https://justindavis882.github.io/Translate-DigitalNative)** 

**To install on a Sidephone:**
1. Open the Live Demo link in your Sidephone's web browser (e.g., Vivaldi or Firefox).
2. Open the browser menu and select **"Add to Home screen"** or **"Install App"**.
3. Launch directly from your home screen for a fullscreen, native utility experience.

## Local Development
Since this is a vanilla stack, no build tools are required. 
1. Clone the repository: `git clone https://github.com/justindavis882/sidephone-translate.git`
2. Open `index.html` in your browser.
